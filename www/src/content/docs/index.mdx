    2. Add `@fiberplane/hono-otel` with two lines of code
        ```ts ins={"Import the instrument function":2-3} ins={"Wrap your application with instrument":11-12}
        import { Hono } from 'hono';

        import { instrument } from '@fiberplane/hono-otel';

        const app = new Hono();

        app.get('/', (c) => c.text('Hello, world!'));

        // ... define your routes


        export default instrument(app);
        ```
    3. Launch FPX in your API directory
        ```sh
        npx @fiberplane/studio@latest
        ```
